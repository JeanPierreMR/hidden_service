Installation

Tor Browser download page

1. Visit: https://www.torproject.org/download/
2. Download the Windows .exe/.dmg/.tar.xz file.
3. Verify the file's signature.
4. When the download is complete, double click the .exe file. Complete the installation wizard process.

Create hidden service

Create the hidden service

To create a hidden service we can use python and Tor

Use a library called stem to bridge between the Localhost and Tor

Install Flask and Stem

pip install flask
pip install stem

Code to open a service on port 5000 of localHost, and create an onion (hidden) route
from stem control import Controller from flask import Flask #setting up the webpage service app = Flask("security_page") port = 5000 host = "127.0.0.1" hidden_svc_dir = "c:/temp/" #here you can add routes @app.route('/') def index(): return "<h1>Este es un mensaje seguro!</h1>" if __name__ == "__main__": print(" * Getting controller") controller = Controller.from_port(address="127.0.0.1", port=9151) try: controller.authenticate(password="") controller.set_options([ ("HiddenServiceDir", hidden_svc_dir), ("HiddenServicePort", "80 %s:%s" % (host, str(port))) ]) svc_name = open(hidden_svc_dir + "/hostname", "r").read().strip() print(" * Created host: %s" % svc_name) except Exception as e: print(e) app.run()
